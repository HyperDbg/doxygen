◆ HvHandleMsrRead()

VOID HvHandleMsrRead ( PGUEST_REGS  GuestRegs)

Handles in the cases when RDMSR causes a vm-exit.

Parameters
GuestRegsGuest's gp registers
Returns
VOID
416 {
417  MSR msr = {0};
418 
419  //
420  // RDMSR. The RDMSR instruction causes a VM exit if any of the following are true:
421  //
422  // The "use MSR bitmaps" VM-execution control is 0.
423  // The value of ECX is not in the ranges 00000000H - 00001FFFH and C0000000H - C0001FFFH
424  // The value of ECX is in the range 00000000H - 00001FFFH and bit n in read bitmap for low MSRs is 1,
425  // where n is the value of ECX.
426  // The value of ECX is in the range C0000000H - C0001FFFH and bit n in read bitmap for high MSRs is 1,
427  // where n is the value of ECX & 00001FFFH.
428  //
429 
430  //
431  // Execute WRMSR or RDMSR on behalf of the guest. Important that this
432  // can cause bug check when the guest tries to access unimplemented MSR
433  // even within the SEH block* because the below WRMSR or RDMSR raises
434  // #GP and are not protected by the SEH block (or cannot be protected
435  // either as this code run outside the thread stack region Windows
436  // requires to proceed SEH). Hypervisors typically handle this by noop-ing
437  // WRMSR and returning zero for RDMSR with non-architecturally defined
438  // MSRs. Alternatively, one can probe which MSRs should cause #GP prior
439  // to installation of a hypervisor and the hypervisor can emulate the
440  // results.
441  //
442 
443  //
444  // Check for sanity of MSR if they're valid or they're for reserved range for WRMSR and RDMSR
445  //
446  if ((GuestRegs->rcx <= 0x00001FFF) || ((0xC0000000 <= GuestRegs->rcx) && (GuestRegs->rcx <= 0xC0001FFF)) || (GuestRegs->rcx >= RESERVED_MSR_RANGE_LOW && (GuestRegs->rcx <= RESERVED_MSR_RANGE_HI)))
447  {
448  msr.Content = __readmsr(GuestRegs->rcx);
449  }
450 
451  //
452  // Check if it's EFER MSR then we show a false SCE state so the
453  // patchguard won't cause BSOD
454  //
455  if (GuestRegs->rcx == MSR_EFER)
456  {
457  DbgBreakPoint();
458 
459  EFER_MSR MsrEFER;
460  MsrEFER.Flags = msr.Content;
461  MsrEFER.SyscallEnable = TRUE;
462  msr.Content = MsrEFER.Flags;
463  }
464 
465  GuestRegs->rax = msr.Low;
466  GuestRegs->rdx = msr.High;
467 }
